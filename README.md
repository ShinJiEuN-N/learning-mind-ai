# Learning Mind AI

AI-based learning self-regulation coach prototype for students who struggle to start studying.

## What It Does

- Checks the student's current learning blocker, mood, energy, and expected focus time.
- Shows a readiness gauge and a short supportive message.
- Generates a small study plan based on learning psychology concepts.
- Provides a focus timer and reflection log.
- Includes a pastel student-friendly UI and dark mode styling.

## Files

- `index.html` - App markup and inline interaction script.
- `styles.css` - Light/dark theme and responsive mobile UI styles.
- `app.js` - Earlier app logic kept for reference while the prototype is being stabilized.

## Run

Open `index.html` in a browser.
